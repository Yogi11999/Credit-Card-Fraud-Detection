# Credit-Card-Fraud-Detection
Machine Learning project to detect fraudulent credit card transactions using Logistic Regression and Random Forest with SMOTE.
# Credit Card Fraud Detection

This project detects fraudulent credit card transactions using Machine Learning.

## Tools Used
Python
Pandas
NumPy
Scikit-learn
Matplotlib
SMOTE

## Models Used
Logistic Regression
Random Forest

## Dataset
https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

## Project Steps
1. Data Preprocessing
2. Handling Imbalanced Dataset using SMOTE
3. Model Training
4. Model Evaluation
5. ROC Curve Visualization

## Results
Random Forest achieved better fraud detection performance with high recall and ROC-AUC score.
